## How to Import the Music Collection

### Option 1: Direct Import Button
1. Open Aetheria in your browser
2. In the left sidebar, look for the **"Import WezClarke Music"** button (blue with sparkles icon)
3. Click the button to open a folder selection dialog
4. Navigate to and select the "Music for Aetheria" folder
5. The app will automatically detect and import all WezClarke tracks

### Option 2: Manual Folder Import
1. Use the standard "Import Folder" button in the sidebar
2. Select the "Music for Aetheria" folder
3. The app will clean up track names automatically
